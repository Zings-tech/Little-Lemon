# Little Lemon Restaurant - Table Booking Web App

This is the Capstone Project for the **Meta Frontend Developer Professional Certificate**. The project is a responsive web application for "Little Lemon," a Mediterranean restaurant, focusing on a functional table reservation system.

## 🚀 Features
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop using CSS Grid and Flexbox.
- **Table Booking Form:** A controlled component form with real-time validation.
- **Dynamic Time Slots:** Uses a Mock API (`fetchAPI`) to retrieve available booking times based on the selected date.
- **State Management:** Implemented using `useReducer` for complex state transitions.
- **Unit Testing:** Robust tests for form validation and initial state using Jest and React Testing Library.
- **Accessibility:** Adheres to WCAG guidelines with semantic HTML and ARIA attributes.

## 🛠️ Built With
- **React** (Functional Components, Hooks)
- **JavaScript** (ES6+)
- **CSS3** (Grid & Flexbox)
- **Jest & React Testing Library** (Unit Testing)
- **Figma** (UI/UX Design)
